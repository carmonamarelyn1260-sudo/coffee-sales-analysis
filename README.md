Coffee Shop Sales Analysis
Sales analysis dashboard for a coffee shop using Power BI and Excel based on 149,000+ transactions.

Project Overview
This project analyzes transactional sales data to understand customer purchasing behavior and product performance in a coffee shop.

The dashboard explores:
- Top revenue-generating products
- Products with the lowest and highest sales volume
- Sales trends by hour of the day
- Revenue distribution by day of the week
- Geographic distribution of sales

Dataset
The dataset contains more than 149,000+ transactions including:
- Product category
- Product type
- Quantity sold
- Unit price
- Transaction time
-Transaction date
- Store location
-Product detail

Key Insights
- Barista Espresso generates the highest revenue.
- Sales peak around 10 am, indicating strong morning demand.
- Monday has the highest revenue among weekdays.
- Some products such as Green beans have very low sales volume.

Tools Used
- Power BI
- Excel
- Data visualization
